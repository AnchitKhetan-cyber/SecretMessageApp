# 🔐 SecretMessageApp

> Built with ❤️ using Kotlin & Jetpack Compose  
> A modern Android application that allows users to send and read hidden messages securely with a clean and minimal user interface.

---

## ✨ Features

- 🔒 Hide secret messages  
- 📩 Decode hidden messages  
- 🎨 Clean and minimal UI  
- ⚡ Fast and lightweight
